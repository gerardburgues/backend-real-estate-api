# Frontend - Streamlit App

This folder contains the Streamlit application for testing the FastAPI endpoints locally.

## Run Locally

```bash
# Make sure FastAPI server is running first
# Then in a separate terminal:
cd frontend
streamlit run app.py
```

## Requirements

The Streamlit app requires:
- streamlit
- requests

Install with:
```bash
pip install streamlit requests
```

Or use the requirements file in the root directory.
